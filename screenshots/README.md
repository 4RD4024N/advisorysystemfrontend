## How to Add Screenshots

1. Take screenshots of your application
2. Save them in this `screenshots/` folder
3. Update README.md with the images:

```markdown
## 📸 Screenshots

### Dashboard
![Dashboard](./screenshots/dashboard.png)

### Documents
![Documents](./screenshots/documents.png)
```

## Tips for Good Screenshots

- Use actual data (not lorem ipsum)
- Show the full UI
- Use consistent window size
- Consider adding annotations
- Optimize image size (keep under 500KB)
